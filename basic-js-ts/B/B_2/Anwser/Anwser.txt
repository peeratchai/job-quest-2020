1. Library ที่คาดว่าจะใช้สำหรับ Project นี้ พร้อมทั้งระบุด้วยว่าทำไมถึงเลือกใช้ เช่น ใช้ lodash สำหรับการจัดการข้อมูล

 -> CSS Framwork ใช้เป็นของ Ant design เนื่องจากมี Components ให้เลือกใช้ได้หลากหลาย และค่อนข้างมี Comunity เยอะ จึงทำให้สามารถ Custom ได้ง่ายยิ่งขึ้น
 -> Axios ใช้ในการ Call API

2.จะต้องจัดการ State ของข้อมูลอย่างไร เพื่อรองรับกับโจทย์นี้ได้

 -> State จะถูกจัดเก็บใน Redux หรือ Context เพื่อให้ง่ายต่อการใช้ข้อมูลในแต่ละ Step 
 -> โดยข้อมูล State จะเป็น Object ที่ระบุ stepName : formData ของ step นั้นๆ เพื่อให้ง่ายต่อการโชว์ Step บนหน้า UI เมื่อมีการ Save draft
 -> จะแบ่งออกเป็น 3 Steps : คือ 1.ข้อมูลส่วนตัว 2.ข้อมูลทั่วไป(การวิ่ง หรือ size เสื้อ) 3.ข้อมูลแพทย์

3.จะใช้วิธีใดในการ Validate Form สำหรับการ Save Draft และสำหรับการ Submit

 -> โดยในแต่ละ Step จะมี 2 ปุ่ม คือ Save draft และ Next (หรือ Submit ใน step สุดท่้าย)
 -> ในการ Save draft จะไม่ทำการ validate ข้อมูล form ว่ากรอกครบหรือไม่
 -> แต่ถ้ากด Next หรือ Submit จะต้องการ Validate form ใน step นั้นๆ ถึงจะสามารถเลื่อน Step หรือ Submit ได้
